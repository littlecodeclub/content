Camp Details

Roblox - Create

$149

OBBIE COURSE: Kids (age 7-12) will learn Building and Publishing games with easy to learn Roblox
Editor. The course will take kids from basics of Roblox building blocks to basic level of creating scripts
using Lua coding. During this learning process, kids will be working with a common goal of making their
own OBBIE course using the roblox toolset, their imagination and our guidance.
Learning Objectives: -
✔ Learn Lua Programming
✔ Use Functions, Commands, and Variables
✔ Create dangerous jumps, a kill block, Spinning Blocks, a Speed Block

Date: Jun 15-18
Time: 9am-12am
Location: Eastmont Middle School, Sandy 

Roblox - Adventure

$149

ADVENTURE MAPS: This course will take kids (age 8-13) from the basics of Roblox building blocks to
basic level of Intermediate scripts using Lua coding. During this learning process, kids will be working with
a common goal of making their own ADVENTURE MAP using the roblox toolset, their imagination and
our guidance. Kids will learn to publish and share their game with their friends.
Learning Objectives: -
✔ Learn Lua Programming
✔ Use Tools, Variables, and the Terrain Editor
✔ Create a Lava Shield, Jump Wand, and Lantern!

Date: Jun 22-25
Time: 9am-12am
Location: Eastmont Middle School, Sandy

Roblox - Royale

$149

BATTLE ROYALE: Kids (age 10-13) will learn Building and Publishing games with easy to learn Roblox
Editor. The course would take kids from basics of Roblox building blocks to basic level of creating scripts
using Lua coding. During this learning process, kids will be working with a common goal of making their
own BATTLE ROYALE using the roblox toolset, their imagination and our guidance.
Learning Objective: -

✔ Create your own Fortnite-style Battle Royale!
✔ Learn Lua Programming
✔ Use Animation, For Loops, Global Scripts, and Client/Server Events

Date: July 13-16
Time: 1 pm-4 pm
Location: East Sandy Elementary, Sandy

Coding - Game Builders Camp

$149

Kids (age 7-12) would build their games portfolio using scratch coding. This camp will take kids from
building simple games to advanced level introducing different concepts of coding on the way. Kids will
have their games portfolio so they can share with their friends and family.

Date: Jun 15 - 18
Time: 1 pm - 4 pm
Location: Eastmont Middle School, Sandy

Coding - Mash-up Camp

$149

Kids (age 7-12) w be experience different fun activites involving coding. Kids would create science
themed scratch games, code a microbit computer and build android app using MIT's APP INVENTOR
tool. The goal of the camp is to provide experience of different application involving coding to bring
interest in STEM activities.

Date: Jun 22 - 25
Time: 1 pm - 4 pm
Location: Eastmont Middle School, Sandy

Coding for Fun Camp

$149

Kids (age 7-12) will learn problem solving, logic building, core math in this fun filled coding camp. Kids will
learn basics of programming by playing mission-based games and using basic coding constructs like
statements, conditions, loops to achieve goals in the games. The course introduces basic syntax of
JavaScript and Python.

Date: July 13 - 16
Time: 9 am - 12 am
Location: East Sandy Elementary, Sandy
